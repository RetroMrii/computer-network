Follow one request through the server

Open a page, watch the server accept the TCP connection, parse the request line, validate the path, choose a MIME type, build an HTTP/1.0 response, and close the connection.

  1. 1 TCP connection accepted
  2. 2 Header bytes buffered until CRLF CRLF

  4. 4 Static path checked for traversal
